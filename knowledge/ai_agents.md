# AI Agents

An AI agent is a system that uses an AI model to plan or choose actions, often with tools such as search, databases, APIs, code execution, or workflow systems. An agent is more than a chatbot when it can take structured steps toward a goal.

Useful agent design starts with a clear objective, bounded tools, memory, evaluation criteria, and safety limits. Beginners should avoid giving agents broad permissions before they understand tool risks and verification.

Agentic AI is strongest when tasks can be decomposed, checked, and corrected. It is weaker when goals are vague, tools are unsafe, or success cannot be measured.

Starter exercise: design a simple research assistant agent with one goal, three allowed tools, and three safety rules.
