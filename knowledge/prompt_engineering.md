# Prompt Engineering

Prompt engineering is the practice of giving an AI model clear instructions, useful context, constraints, examples, and a success standard. Strong prompts reduce ambiguity and help the mentor produce more useful answers.

A practical prompt structure is: role, task, context, constraints, output format, and quality bar. For example: "Act as a technical mentor. Review this n8n workflow. Explain mistakes for a beginner. Return strengths, risks, and next steps."

For learning, prompts should ask the model to explain reasoning in beginner-friendly language, define unfamiliar terms, and give a small practice task. Avoid asking for a final answer only when the goal is skill development.

Starter exercise: rewrite a vague prompt into a structured prompt with role, task, context, constraints, and output format.
